Objectives

During this assignment you should learn:

  • How to assign names to variables
  • The different types of variables and how to handle them
  • The basics of debugging a prgram and readin error messages

Assignment

Please write the Python for the following questions. The file you submit should be saved as a .py file using Visual Studio Code. All the exercises in this assignment can be included within a single file.

  1. Exercise 1: Input a String
    Please write a program to ask the user to input their name and then print out a greeting to the user that includes their name.
    Example Output -
    "Hello! What is your name? "
    "Hi, John"
  2. Exercise 2: Input and convert an integer
    Write a program that converts a dog's age from actual years to human equivalent years. (Each year a dog lives is approximately 7 human years so a 3 year old dog is 21 in human equivalent years.) Format the input and output so it is 'user friendly.'
  3. Exercise 3: Input and manipulate floating point numbers
    Write a program to that computes the area of a rectangle. Have the user enter the length and the width and then print out the resulting area. Make sure your program is 'user friendly' by properly labeling the input and output. Can your program handle floating point numbers as well as integers?
  4. Exercise 4: Challenge Exercise, you will need to save an manipulate many variables
    Write a program to compute the actual tax amout based on the yearly income that an individual enters using the following tax table.
    • The first $20,000 that an individual makes is taxed at 0%
    • Income between $20,001 and $50,000 is taxed at 10%
    • Income between $50,001 and $75,000 is taxed at 15%
    • Income between $75,001 and $100,000 is taxed at 20%
    • Income between $100,001 and $150,000 is taxed at 25%
    • Income above $150,000 is taxed at 30%
    There are many ways to solve this problem. If you are familiar with if statements that is an option, but unecessary. You can instead use the max function - max(number, 0) will return the the number if it is positive, otherwise it will return zero. (This exercise is basically a weighted average.)
  5. Exercise 5: Debugging
    Using Python comments describe any errors you encountered while writing your program and what steps you took to correct them.
