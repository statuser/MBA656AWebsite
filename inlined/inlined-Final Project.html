Assignment

Your assignment is to prepare a report that summarizes the findings in the previously collected survey. You will have 15 minutes to present your results. The survey is fairly short. Do not feel the need to fill all 15 minutes if the data does not warrant that. The most important learning from this assignment is to understand the core business problem that to be addressed and finding a solution using the data provided.

In keeping with the topics in the course, please you Python for the data analysis. There are many other tools available that are useful and helpful including analytics built into Qualtrics. For the purposes of this assignment please use Python however. (The main exception is that sometimes I find it best to use the visualization tools built into the presentation software for the final visualizations.

Make sure your presentation includes the following:

  • A statement of the business question that motivated the project
  • The answer to that business question
  • Visualizations and data summaries to support your solution
  • Conclusions and expected results
  • Appendix with data and code
