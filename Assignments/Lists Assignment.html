The exercise for today is to write a computer program to demonstrate a core statistical concept called the Central Limit Theorem. The Central Limit Theorem states that the distribution of the mean of any set of equivalent random samples is distributed as a normal distribution. This theorem can be proved mathematically, but it is fairly easy to demonstrate it with some basic computation.

You assignment is to create a python program to demonstrate the central limit theorem. You should be able to do this using only the concepts we have talked about already in class. You should not use any additional libraries except for the "random" package and a package for plotting the results called "matplotlib".
