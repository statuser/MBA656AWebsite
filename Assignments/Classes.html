Objectives

  • Practice calling and using classes and objects

  • Continuing to build computational thinking skills by programming to solve new problems

Assignment

Your assignment involves creating a series of drawing using the Turtle class built into Python. You will need to provide code to complete each of the programing tasks. You will need a separate program for each problem since you can only call turtle.mainloop() once for each program.

Your program shoudl do the following:

  • Create a 5-sided star. (Hint: The angle for each point of the star is 144 degrees.)
  • Draw a pair of circles that do not overlap. (Hint: You will need to look at the help using help(Turtle) for more information on different methods available.
  • Write a function that will create a shape with an arbitrary number of sides. Call this function with multiple different sides. (Hint: You will probably need to pass a turtle instance to this function.)
  • Challenge: Write code to draw a smiley face.

Remember that you will need to call the turtle.mainloop() function at the end of each program.

Deliverables

Please submit the assignment on Canvas. You will need to submit the four code files for this assignment, one for each problem.
