Objectives

  • Practice building a module that can be reused in other programs
  • Use the "__name__" variable to make sure test code is available
  • Practice refactoring code for reusability

Assignment

Your assignment is to convert the code you wrote in the functions assignment into a module. This module should be usable in different programs you write using a simple import statement. At the same time your code should maintain the ability to run the test code that you have previously developed. You will need to modify your code to handle both cases.

Make sure your code includes the following:

  • Your module file, modified to run either independently or as an imported module
  • A separate test program that imports your module and calls each of the functions
  • Before submitting test your code both as a stand alone program and as a module

Deliverables

Please submit the assignment on Canvas. You will need to submit two files for this assignment

  1. Your utility functions file
  2. A file that imports the utility functions file and calls each function
