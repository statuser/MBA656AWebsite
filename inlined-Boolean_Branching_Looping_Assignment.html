Rock, Paper, Scissors

Rock, paper, scissors is a classic game that has been used to decide everything form multi-million dollar investments to who gets the front seat. You assignment is to create a computer opponent to play rock, paper, scissors against. This program will need to:

  • Randomly select either rock, paper, or scissors
  • Wait for the player to input their selection
  • Verify that the users selection is valid
  • Determine if there is a tie and if there is restart the game
  • Announce what each participant threw and who the winner is.

Remember that for this introductory class you can assume users will not intentionally try to break the rules and as such do not need to make extensive input checks.

Sample output for the program looks like: 

	  
	  Hello, let's play Rock, Paper, Scissors!
	  I have picked my hand.
	  Please type in "rock", "paper", or "scissors" to select your hand?
	  **User types in**rock
	  I selected paper, I win!
