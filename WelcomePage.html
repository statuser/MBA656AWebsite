Tips for Success

The following tips will help you succeed in this class

  • To ensure you don't lose work, it is recommended that you work offline using a text editor such as Visual Studio Code and then copy/paste your post into the course's submission forms.
  • Make sure you keep up with the class lectures and assignment. They build on each other and it will be very hard if you fall behind
  • Set up your Canvas notifications and check Canvas regularly. Canvas is the source of truth regarding due dates, assignments, policies, and announcements
